Nav-bar

Navbar doc link# If you want to link to a specific doc, this special navbar item type will render the link to the doc of the provided docId. It will get the class navbar__link--active as long as you browse a doc of the same sidebar.

docusaurus.config.js;
module.exports = {
themeConfig: {
navbar: {
items: [
{
type: "doc",
docId: "introduction",
//// Optional
position: "left",
label: "Docs",
activeSidebarClassName: "navbar__link--active",
docsPluginId: "default",
},
],
},
},
};
